# LabOps Actions runner

This image extends the pinned GitHub Actions runner with the pinned `gh`, `jq`,
and `kubectl` clients required by the shared labtest deployment workflow. The
generic Docker workflow publishes the semantic version from `VERSION` only on
`master`; pull requests receive development and commit-SHA tags instead.

The image contains no credentials. ARC mounts the GitHub App authentication and
the least-privilege labtest kubeconfig from Kubernetes Secrets generated by the
Bitwarden operator at runtime.
